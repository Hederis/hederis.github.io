How it works

With Hederis, you can upload a Microsoft Word manuscript file and get professionally laid-out print and ebook files in a flash. Our design and layout interface allows you to create beautiful, custom book designs for every project. Adjust your page layout right in the browser, and export your project to PDF, EPUB, Kindle-optimized EPUB, HTML, and an up-to-date Microsoft Word file that includes all your changes.
