Press release: Tech startup launches cloud-based bookmaking program for trade publishers.

[Portland, Oregon, July 29, 2020] – Hederis, Inc., a tech startup creating software for trade publishers, has launched their AI-supported book design app.

The Hederis app is a collaborative cloud-based program—meaning editors, designers and project managers can easily access the latest version of a book’s text and design throughout its production cycle.

What makes this app stand out is the speed with which a user can create a high-quality book layout without the need for coding experience. Made specifically for books, Hederis combines automation and hands-on layout tools, so that users can create laid-out books with the click of a button using the Hederis design templates, or create their own design and adjust the layout of each page in the book.

After a successful Kickstarter campaign in 2018, early book prototypes were built using Hederis. Over 500 copies were printed for more than 190 campaign backers. Hederis began beta testing the software in 2019, with partners Erewhon Books. Together they produced Erewhon’s first title (THE FORTRESS) just before COVID, with 6 more titles currently ramping up production.

About the company:

Hederis is redesigning automation for human intervention. We believe publishing tools should be accessible and efficient for everyone.
